# AI Interview Trainer (OpenEnv)

## Overview
This project simulates a real-world interview environment using OpenEnv.

## Features
- 3 difficulty levels (easy, medium, hard)
- Reward-based evaluation
- Deterministic baseline agent
- Optional AI evaluation (Groq API)

## How to Run

### Baseline
python inference.py

### AI Mode
python ai_demo.py

## Reward Logic
- Depth (length)
- Reasoning
- Example usage

## Deployment
Compatible with Docker & Hugging Face Spaces
